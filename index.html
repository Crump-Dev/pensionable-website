Defensible by design

Five structural answers. Not policy promises.

01
The scheme and their advisors retain ownership of the deliverable

Every interpretive call and sign-off stays with the actuary. The engine accelerates; it does not decide.

02
No LLM in the runtime path

Deterministic code throughout. The LLM builds the calculator. It never enters a live calculation.

03
Every output regression tested

Every rebuild is fully regression tested and documented in easy-to-read reports, delivered as documents or self-contained websites.

04
Two independent model families reviewing each other, orchestrated by a highly experienced human DB analyst

One model executes. A second, from a different vendor, audits with no shared context. A senior DB analyst orchestrates both. Structurally analogous to peer review, with human expertise at the centre.

05
Anonymisation before any LLM access

Personal data never reaches a model. A six-phase pipeline strips and substitutes locally before any LLM sees the artefact.
